Простая веб‑страница с информацией о герое «Король варваров» из игры Clash of Clans

## Описание.
1.собрать и структурировать данные о навыках героя;
2.описать условия разблокировки и особенности прокачки;
3.предоставить рекомендации по тактическому применению героя в атаках и защите;
4.дать ссылки на полезные внешние ресурсы.
5.обеспечить интуитивно понятный интерфейс для быстрого поиска нужной информации.

## Особенности 
ИНФО_блок «Обо мне»: содержит краткое описание роли и предназначения Короля варваров в игре, сопровождается изображением персонажа.
Раздел «Навыки»: нумерованный список со всеми способностями героя, где каждая позиция включает название (выделено жирным) и подробное описание эффекта

## Технологии 

Список технологий, библиотек и инструментов, использованных в проекте:
Язык разметки: HTML5 — для структурирования контента (заголовки, списки, абзацы, ссылки, изображения).
Язык стилей: CSS — для визуального оформления страницы (цвет фона, ш цвета ссылок, отступы).
